## 목표

- **DB 내부 동작 원리 이해**: MySQL의 아키텍처와 엔진 동작 원리를 깊이 있게 파악한다.
- **실험 기반 문제 해결 경험 축적**: "왜?"라는 질문에 대해 실제 데이터와 지표로 답을 찾는다.
- **성능 최적화 능력 키우기**: 쿼리 실행 계획을 분석하고 병목을 해결하는 능력을 기른다.

## 학습 방식

> [RealMySQL 8.0](http://www.yes24.com/Product/Goods/103415627) 책을 바탕으로 학습 및 실습 진행
>

- 각 주차 주제에 맞춰 **이론 학습 + 실습**으로 진행한다.
- 실험 결론을 도출할 때, 이론 학습 내용 바탕으로 작성한다.
- 이론 학습 내용, 실습 방법은 Github에 정리한다.

## 운영 규칙

- 수업은 **매주 화요일 16시**에 1시간 진행한다. (최대 30분 연장 가능)
- **선생님**
    - 스터디 전까지 주제에 맞는 챕터를 골라 공부하고, 수업 내용을 정리한다.
    - 학생들이 읽어와야 할 챕터를 주말 전까지 공지한다.
    - 실습이 필요하다면, 이론 내용을 바탕으로 미리 실험을 설계하고 진행하여 수업 때 차질이 없도록 한다.
    - 스터디 후, 이론 요약과 실험에 대해 파일로 정리해 PR을 제출한다.
    - 순서 : 라이 → 봉구스 → 티모 → 서여
- **학생**
    - 수업 전까지 선생님이 정한 챕터를 읽는다.
